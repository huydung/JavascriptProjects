History & features

Images Hunter is a simple Javascript application which help you find images on the web faster and also quickly crop/resize them to an exact dimensions without the need of an additional image editor.

In my current position, one of my weekly tasks is to post articles to the company's website (I developed that website too), and I have to search for suitable images to be used as thumbnail of the posts, and then crop/resize them to the exact dimensions of 200x120 pixels. It takes time. Approximately 30 minutes for 7 posts. With this application, I saved half of the time. Hope it can helps you too!

Please keep in mind that, you have responsibility to only use images that their authors allowed.
